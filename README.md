## How to use:

1. `git clone https://github.com/rolling-scopes-school/loori-r-2018Q3.git`
2. `cd loori-r-2018Q3/rss-mentor-dashboard`
3. `npm install`
4. `npm server/check.js`
5. `follow the instructions for authorization`
6. `npm start`
